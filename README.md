# Furchain Protocol

JSON Schemas for the Furchain Protocol v1.

## Schemas

| Schema | URL |
|--------|-----|
| Repository manifest | `https://furchainlabs.github.io/protocol/v1/furchain.schema.json` |
| Public key registry | `https://furchainlabs.github.io/protocol/v1/keys.schema.json` |
| Character definition | `https://furchainlabs.github.io/protocol/v1/character.schema.json` |
| Publication envelope | `https://furchainlabs.github.io/protocol/v1/publication.schema.json` |
| Subscription groups | `https://furchainlabs.github.io/protocol/v1/group.schema.json` |

## Usage

Reference a schema in your JSON files:

```json
{
  "$schema": "https://furchainlabs.github.io/protocol/v1/furchain.schema.json"
}
```
